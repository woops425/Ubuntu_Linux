리눅스 기본 명령어

ls : Windows의 dir과 같은 역할로, 해당 디렉토리(폴더)에 있는 파일의 목록을 보여줌
ex) # ls /etc/systemd
ls -l : 현재 파일 목록이 쭉 뜸

cd : 디렉토리를 이동
ex) # cd../etc/systemd

pwd : 현재 디렉토리의 전체 경로를 출력

touch : 크기가 0인 새 파일 생성.
ex) touch abc.txt

rm : 파일이나 디렉토리를 삭제
ex) rm abc.txt
rm -rf abc(이렇게 삭제하면 안에 뭐가 들었든 삭제하기 때문에, 복구가 불가함)

cp : 파일이나 디렉토리를 복사
ex) cp abc.txt bcd.txt

mv : 파일과 디렉토리의 이름을 변경하거나 위치 이동 시 사용. 이름을 바꾸면서 위치 이동을 동시에도 가능
ex) mv abc.txt www.txt
mv bbbb.txt /etc/cccc.txt

mkdir : 새로운 디렉토리를 생성
ex) mkdir abc
한 번에 여러 개의 디렉토리를 생성할 때에는,
mkdir -p bcd/efg/hij

rmdir : 디렉토리 삭제. 단, 디렉토리가 비어 있어야 함
ex) rmdir abc

cat : 텍스트로 작성된 파일을 화면에 출력
ex) cat /etc/systemd/user.conf

head/ tail : 텍스트로 작성된 파일의 앞 10행 또는 마지막 10행만 출력
ex) head /etc/systemd/user.conf
만약, 5줄만 보고싶다?
ex) head -5 /etc/systemd/user.conf

more : 텍스트로 작성된 파일을 화면에 페이지 단위로 출력. 스페이스바를 누르면 다음 페이지로 이동, Q를 누르면 탈출
e) cat /etc/systemd/user.conf | more

less : more과 비슷한 기능을 수행하나, 키보드 방향키로 위 아래 이동이 가능하고, 좌우로 페이지 넘김도 가능해서 more 보다 보기 편함

file : File이 어떤 종류의 파일인지를 표시
ex) file /etc/systemd/system.conf

gedit : 리눅스에서의 메모장 같은 개념
gedit 을 입력하면 메모장이 켜지고, 저장한 파일을 열고 싶을 땐, gedit test1.txt 꼴로 입력하면 됨

nano: gedit과 비슷. nano nano1.txt 꼴로 열 수 있고, 컨트롤 + X 로 나갈 수 있음. 저장은 y

파일과 디렉토리의 소유와 허가권

chmod : 파일 허가권 변경 명령어
ex) chmod 777 sample.txt

chown은 root 권한이 있는 사용자만 사용할 수 있는 명령어.
chown(파일 소유자를 바꿀 수 있음)
chgrp(파일 소유 그룹을 바꿀 수 있음)
ex) chown ubuntu(소유자):ubuntu(소유 그룹)
sample.txt 또는
chown ubuntu sample.txt alc
chgrp ubuntu sample.txt

프로그램 설치를 위한 dpkg
설치 : dpkg -i 패키지파일이름.deb
삭제 : dpkg -r 패키지이름
또는 dpkg -P 패키지이름 <- 설정파일까지 삭제
패키지 조회
dpkg -l 패키지이름 <- 설치된 패키지에 대한 정보 보여줌
dpkg -L 패키지이름 <_ 패키지가 설치한 파일 목록을 보여줌
아직 설치되지 않은 deb 파일 조회
dpkg --info 패키지파일이름.deb
-> 패키지파일에 대한 정보를 보여줌

dpkg 의 의존성문제를 해결하기 위해 apt가 등장

편리한 패키지 설치를 위한 apt
apt 명령 : dpkg 명령의 패키지 의존성 문제를 완전하게 해결. a를 설치하기 위해 b,c,d가 필요하다면, 이를 알아서 자동으로 다운로드해서 설치하는 방식

기본 사용법
기본 설치 : apt -y install 패키지이름
("-y"는 사용자 확인을 모두 yes로 간주하고 설치를 진행한다는 옵션)
패키지 목록 업데이트 : apt update
삭제 : apt remove/purge 패키지 이름
사용하지 않는 패키지 제거 : apt autoremove
내려 받은 파일 제거 : apt clean 또는
apt autoclean
