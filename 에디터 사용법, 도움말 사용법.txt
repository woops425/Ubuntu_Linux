에디터 사용법, 도움말 사용법

vi 입력 시, vi 화면으로 들어감
쉬프트 + ' : ' 을 입력하고, q 입력 후 엔터를 누르면 빠져나옴.

vi new.txt <- new.txt 라는 새로운 파일을 만들면서 vi 화면으로 들어감.
이 때, i (insert)또는 a(append)를 누르면 입력 모드로 바뀜(왼쪽 아래 끼워넣기로 바뀜)

입력 모드를 마치려면 esc를 눌러 다시 명령 모드로 돌아가게 되면, 끼워넣기가 사라지고
쉬프트 + 콜론(:)을 누른 뒤, wq(write quick)을 입력 후 엔터를 누르면 저장이 완료되고 나가짐.
입력 혹은 수정 내용을 저장하지 않고 나가고 싶다면, q! 를 입력하면 됨

vi 작업 중, 터미널을 닫거나 하는 등으로 저장을 하지 않고 비정상적인 종료를 했을 때, 만약 이 파일을 복구할 필요가 없다면,
ls -al (숨김파일까지 모두 보여줌. 숨김파일은 .~~~로 시작)
거기서, 만약 파일명이 new.txt였다면,
rm -f .new.txt.swp 입력 시, 파일 삭제 후 다시 정상적으로 vi 사용 가능
