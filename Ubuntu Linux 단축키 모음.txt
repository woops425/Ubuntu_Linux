Ubuntu Linux 단축키 모음

*#프롬프트는 root 사용자
*$프롬프트는 일반 사용자

(좌)Ctrl + (좌)Alt : 마우스 활성화
Ctrl + ALT + Enter : 전체화면 끄기
쉬프트 + 스페이스바 : 한/영 변환
tab : 다음 키
쉬프트 + 탭 : 이전 키

가상 콘솔 이동 단축키 : 컨트롤 + 알트 + F2 ~ F6
(컨트롤+알트+F2 는 X윈도우 모드)
명령어로도 이동 가능 : chvt 콘솔번호
ex) 3번 모니터로 이동 -> chvt 3

root 사용자일 때,
shutdown -h +5 명령어 : 5분 뒤 컴퓨터 종료
shutdown -c : 셧다운 명령 취소
shutdown -k +10 : 사용자들에게 10분 뒤 컴퓨터가 종료된다고 알림. 실제로 종료되진 않지만, 사용자 입장에서는 이를 구분할 수 없음
